This is app showing all car datas with cardview. coded using IDE Android Studio 2.3 and with following technologies: 

1. MVP Pattern
2. Dagger 2
3. RxJava 2
4. Retrofit 2
5. Glide 
6. ORM GreenDao

For automation UI and unit test, this app using following technology:
1. Espresso
2. Mockito
3. Robolectric
